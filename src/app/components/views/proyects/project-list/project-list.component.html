Nakama Store Cba

Aplicación web de tipo e-commerce realizada para una tienda de disfraces.

Funciones principales:

  • Registro e inicio de sesión de usuarios.
  • Panel de gestión para usuarios administradores.
  • Venta de productos a través de un sistema intuitivo y seguro.

Sistema Despensa "Moe"

Aplicación de escritorio de tipo ERP (sistema de gestión comercial) desarrollado para una despensa.

Funciones principales:

  • Gestión de productos, incluyendo precios, stock y promociones (permanentes y temporales).
  • Gestión de ventas, incluyendo fecha y hora, detalles, costos y ganancias.
  • Apertura, cierre y contabilización de caja.
